Acheter le livre

JavaScript Éloquent est un livre fournissant une introduction au langage de programmation JavaScript et à la programmation en général.

Un mélange concis et équilibré de principes fondamentaux et de détails pratiques. J'ai aimé le développement de programmes sur le mode du tutoriel et du jeu. Ce livre a ravivé mes tout premiers plaisirs de la programmation. Plus, du JavaScript !
—Brendan Eich, l'homme qui nous a donné JavaScript

Le livre existe sous deux formes. Il a originellement été écrit et publié sous forme numérique, celle-ci incluant des exemples interactifs et un mécanisme pour jouer avec tout le code d'exemple. Cette version est distribuée sous une licence libre.

J'ai publié une version révisée du livre au format papier. La structure de cette version reste largement la même, mais tout le texte a été complètement modifié et mis à jour. Vous pouvez le commander sur Amazon ici. Il y a toujours un environnement de développement pour cette version, sur une page séparée. Les errata sont ici.

Pour en savoir plus sur la traduction de ce livre en français, consultez la page des contributeurs.

La version numérique est disponible sous les formats suivants :

  • HTML
  • HTML, zippé
  • HTML, une seule page (non interactif)
