JavaScript Éloquent : Errata

Une version physique de cette traduction en français du livre JavaScript Éloquant est en projet. Une fois celle-ci publiée, vous pourrez voir ici les erreurs trouvées dans la version papier de cette traduction après sa publication.

Cependant, seule la version originale anglaise existe au format papier pour le moment, et nous vous conseillons d'aller voir sur la page consacrée du site officiel de cette version pour y trouver les erreurs présentes dans sa première édition ou en signaler une nouvelle (toujours pour dans la version anglaise).
