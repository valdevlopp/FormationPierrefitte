JavaScript Éloquent Bac à sable pour le code de l'édition papier [help]

La console ci-dessous permet de modifier et d'exécuter du code JavaScript. La sortie (des functions show et print) s'affiche dans la zone à gauche. Certaines valeurs peuvent être cliquées pour les inspecter plus en détails. La ligne en dessous peut être utilisée pour écrire des expressions JavaScript et les évaluer à la volée. A droite, on trouve l'éditeur dans lequel le code peut être saisi ou chargé, et executé en utilisant les boutons situés en dessous.

La liste 'Chapitre X' en bas de la page contient presque tout le code présent dans le livre. Cliquez sur le triangle pour développer ou réduire la liste des morceaux de code d'un chapitre. Puis cliquez sur le nom d'un morceau de code pour le charger, ou cliquez sur [exécuter tout] pour exécuter tout le code d'un chapitre. Pour les chapitres avec de gros exemples, ce dernier choix est le moyen le plus facile de voir les exemples en action.

En bas à droite de l'écran se trouve une fenêtre séparée dans laquelle le code de la console s'exécute directement. Les exemples de certains des chapitres (comme le HTML créé dans le chapitre 5) s'afficheront ici.
